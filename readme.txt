Just another scroll or click to top button plugin. Simple but flexible.

== Description ==

Click to Top Button is a WordPress plugin that adds a convenient and customizable button to quickly scroll back to the top of a page.

== Features ==

* No setup necessary. Just install and activate.
* Unlimited color selection.
* Rounded corners.
